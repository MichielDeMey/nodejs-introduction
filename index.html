## error-first callbacks * used in Node core * plenty of libraries * The **first argument** â†’ error object * The **second argument** â†’ response data
Merge the contents from 2 files ```javascript const fs = require('fs') const async = require('async') async.parallel([ callback => fs.readFile('foo.txt', 'utf8', callback), callback => fs.readFile('bar.txt', 'utf8', callback) ], (error, results) => { if (error) return console.error(error) console.log(results.join('\n')) }) ``` `node merge-callback.js`
Merge the contents from 2 files (Promise edition) ```javascript const fs = require('fs') const pify = require('pify') const readFile = pify(fs.readFile) Promise.all([ readFile('foo.txt', 'utf8'), readFile('bar.txt', 'utf8') ]) .then(results => { console.log(results.join('\n')) }) .catch(error => { console.error(error) }) ``` ``node merge-promise.js``
